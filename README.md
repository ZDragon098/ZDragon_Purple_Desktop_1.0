# 💜 ZDragon Purple Desktop Setup

Giao diện máy tính tối giản với tông màu tím nhạt, sử dụng Rainmeter,  Windhawk và Lively Wallpaper.

![Desktop Preview](Youtuber: Jaymei https://www.youtube.com/watch?v=Kvkl7RvqkmU)

## 🛠 Yêu cầu hệ thống và app cần tải
* **OS:** Windows 11.
* **Ram:** Tối thiểu *8GB*.
* **1. Rainmeter:** Phiên bản 4.5 trở lên.
* **2. Windhawk:** Để tùy biến thanh Taskbar.
* **3. Lively Wallpaper:** Để làm hình nền động.

## 🚀 Cách cài đặt
1. Tải file `Purple_Desktop_1.0.rmskin`.
2. Chạy file để Rainmeter tự động cài đặt các skin:
    * **Music:** Widget trình phát nhạc (Medium size).
    * **Time & Calendar:** Widget thời gian và lịch.

3. Mở **Windhawk** và cài đặt các bản mod.
    * **Windows 11 Taskbar Styler:** Chỉnh setting > Matter.
    * **Windows 11 Start Menu Styler:** Chỉnh setting > Liqid Glass (for the redesigned Start Menu).

## Lưu ý
1. Nhớ để thanh taskbar ở vị trí center.
2. Để thanh Search ở dạng icon only.

## 📝 Lưu ý cho Music Skin
Để widget nhạc hoạt động, hãy đảm bảo bạn đã chỉnh sửa file `Variables\Music.inc` để khớp với trình phát nhạc bạn đang dùng (Spotify, Web, v.v.).
